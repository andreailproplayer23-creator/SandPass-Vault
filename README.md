# ⏳ SandPass - Temporary Vault & Generator

![Python](https://img.shields.io/badge/Python-3.9+-blue?style=for-the-badge&logo=python)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)
![Security](https://img.shields.io/badge/Security-Critical_Alert-red?style=for-the-badge)

**SandPass** è un generatore di password temporanee progettato per chi mette la sicurezza al primo posto. Crea password robuste che si "distruggono" automaticamente dopo un tempo prestabilito, venendo eliminate istantaneamente sia dagli appunti (Clipboard) che dall'interfaccia dell'app.

---

## 🌟 Funzionalità Principali

* **🛡️ Vault Temporaneo**: La tua password ha una data di scadenza. Scegli tu quanto deve durare (da 5 a 120 secondi).
* **⚠️ Allarme Critico**: Un avviso di sistema (con suono di errore) ti avverte nel momento esatto in cui la password viene distrutta.
* **🥷 Stealth Mode**: Chiudi la finestra e l'app continuerà a proteggerti dalla *System Tray* (vicino all'orologio).
* **⚙️ Custom Generator**: Controllo totale su lunghezza, simboli e numeri per soddisfare ogni requisito di sicurezza.
* **📋 Auto-Clear Clipboard**: Pulizia profonda degli appunti di Windows allo scadere del timer per evitare incollaggi accidentali.

---

## 🚀 Come Installare

Per utilizzare **SandPassVault** sul tuo PC, segui questi semplici passaggi:

1. Vai nella sezione **[Releases](https://github.com/tuo-username/SandPass-Vault/releases)** di questo repository.
2. Scarica l'ultimo file **`SandPassVault_Setup.exe`**.
3. Avvia l'installer e segui le istruzioni a schermo.

> **Nota:** Si consiglia vivamente di utilizzare l'installer ufficiale per garantire che le icone della barra di sistema e le notifiche critiche funzionino correttamente.
